What is Java Script Object?

In JavaScript, arrays are objects, functions are objects,regular expressions are objects, and, of course, objects are objects.

What an object can be?

An object is a container of properties, where a property has a name and a value. A property name can be any string, including the empty string. A property value can be any JavaScript value except for undefined.

Why Objects?

Objects are useful for collecting and organizing data. Objects can contain other objects, so they can easily represent tree or graph structures.

Features
  • Allows one object to inherit the properties of another object.
  • Key value pair data representation
  • Can store array and object itself
Object Literals

Object literals provide a very convenient notation for creating new object values. An object literal is a pair of curly braces surrounding zero or more name/value pairs.
