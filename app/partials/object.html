What is JavaScript Object?

In JavaScript, arrays are objects, functions are objects,regular expressions are objects, and, of course, objects are objects. Almost "everything" in JavaScript are treated as objects.

What an object can be?

An object is a container of properties, where a property has a name and a value. A property name can be any string, including the empty string. A property value can be any JavaScript value except for undefined.

Why Objects?

Objects are useful for collecting and organizing data. Objects can contain other objects, so they can easily represent tree or graph structures.

Different ways to store data in JavaScript Object

There are different ways to store data into JavaScript object, here are couple of examples.

  • Object containing object

    In JavaScript one object can contain other objects as well. 

        <script type="text/javascript">
        var flight = {
            airline: "Air India",
            number: 815,
            departure: {
                IATA: "SYD",
                time: "2004-09-22 14:55",
                city: "Delhi"
            },
            arrival: {
                IATA: "LAX",
                time: "2004-09-23 10:42",
                city: "Banglore"
            }
        };
    </script>

  • Object containing Array

    As we talked earlier an Array is also treated as object and object can store other objects. 

        <script type="text/javascript">
        var empty_object = {};
        var company = {
            "name": "Sapient",
            "location": "Gurgaon",
            "employees" : [
                {'id':10,'name':'Deepak Sharma'},
                {'id':20,'name':'Rajat Malhotra'},
                {'id':30,'name':'Ajay Verma'}
            ]
        };
    </script>

  • Object containing Methods

    Methods are the actions that can be performed on objects.

        <script type="text/javascript">
        var empty_object = {};
        var company = {
            "name": "Sapient",
            "location": "Gurgaon",
            getEmployees : function(){
                return 2000;
            },
            getProjects : function(){
                return ["Project 1","Project 2","Project 3","Project 4","Project 5"]
            }
        };
    </script>

Object Retrieval

We can retrieve object properties by two ways

  • Using (.) dot notation 
        <script type="text/javascript">
        var firstName = person.firstName,
            lastName = person.lastName; 
    </script>
  • Using "[]" bracket notation 
        <script type="text/javascript">
        var firstName = person['firstName'],
            lastName = person['lastName']; 
    </script>
