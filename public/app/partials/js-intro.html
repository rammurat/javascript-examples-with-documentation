What is JavaScript?

JavaScript is a programming language used to make web pages interactive, and increasingly considered an "assembly" language (a compiler target) according to its creator.

It is most commonly used as part of web browsers, whose implementations allow client-side scripts to interact with the user, control the browser, communicate asynchronously, and alter the document content that is displayed.

It is also being used in server-side network programming (with Node.js),game development and the creation of desktop and mobile applications.

JavaScript is classified as a prototype-based scripting language with dynamic typing and has first-class functions. This mix of features makes it a multi-paradigm language, supporting object-oriented, imperative, and functional programming styles.The syntax of JavaScript is actually derived from C and rest of its own.

What can we do with JavaScript?

  • Put text in an HTML page on-the-fly.
  • Change HTML elements
  • Delete HTML elements
  • Create new HTML elements
  • Copy and clone HTML elements
  • Update attributes
  • Detect visitors' browsers.
  • Create cookies
  • Validate web form data

What do I need to run JavaScript?:

How to include JavaScript into your Web page?

There are two ways to includes JavaScript in to the page

  • Internal Script

    While using internal JavaScript, As per W3C Standard and Consortium we should write JavaScript in head section under script tag, but lot of experienced JavaScript geeks don't prefer that and suggest to write/include your JavaScript at the bottom of the page as sometime due to heavy JavaScript, html takes more time to render on the page and its not good User experience.

    Here is an example of one dummy index.html web page.
    <!Doctype>
<html>
    <head>
        <script type="text/javascript">
            var empty_object = {};
            var person = {
                "first-name": "Deepak",
                "last-name": "Sharma"
            };
        </script>
    </head>
    <body>
        <p>This is web page</p>
    </body>
</html>

  • External JavScript File

    You can include JavaScript externally by putting JavaScript code into seperate file and then include into your web page.

    Here is an example of one dummy index.html web page and corresponding JavaScript script.js file.
    <!Doctype>
<html>
    <head>
        <script type="text/javascript"src="script.js">   </script>
    </head>
    <body>
        <p>This is web page</p>
    </body>
</html>

JavaScript VS Java

The JavaScript programming language, developed by Netscape, Inc., is not part of the Java platform.

JavaScript does not create applets or stand-alone applications. In its most common form, JavaScript resides inside HTML documents, and can provide levels of interactivity to web pages that are not achievable with simple HTML.

Key differences between Java and JavaScript:

  • Java is an OOP programming language while JavaScript is an OOP scripting language.
  • Java creates applications that run in a virtual machine or browser while JavaScript code is run on a browser only.
  • Java code needs to be compiled while JavaScript code are all in text.

Good Parts

  • JavaScript is the most popular programming language in the world.
  • JavaScript can manipulate the DOM (change HTML contents).

Bad Parts

  • Performance gets down if script is too heavy.
