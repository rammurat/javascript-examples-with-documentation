What is (this) in JavaScript?

In JavaScript this always refers to the "owner" of the function we're executing, or rather, to the object that a function is a method of. When we define our faithful function doSomething() in a page, its owner is the page, or rather, the window object (or global object) of JavaScript.

Scope and behaviour of (this)

this always refers to current scope and it has different scope with methods,objects and classes. Here are couple of examples with live demo.

  • (this) by default

    By default this denotes to window level object whether you access this directly in the page or call under any method. 

    <script type="text/javascript">
	// define a function
	var myFunction = function () {
	  console.log(this);
	};

	// call it
	myFunction();
</script>

  • (this) and Object

    When we access this with in a object it returns object itself.In below example method name calls itself and access other members value using this keyword.

    <script type="text/javascript">
	var person = {
		firstName: "Deepak",
		lastName: "Sharma",
		name :function(){
			return this.firstName + " " + this.lastName;
		}
	};
</script>

  • (this) and Constructor

    You can use this with Constructor where you can access any method of an object without using return keyword.

    <script type="text/javascript">
	var myConstructor = function () {
		this.someMethod = function () {
			console.log(this);
		};
	};

	var a = new myConstructor();
	a.someMethod();
</script>

  • (this) and Prototype

    You can apply the same concept on Prototype as well.here is quick example:

    <script type="text/javascript">
	var myConstructor = function () {
		this.someMethod = function () {
			console.log(this);
		};
	};

	myConstructor.prototype = {
		somePrototypeMethod: function () {
			console.log(this);
		}
	};

	var a = new myConstructor();
	a.someMethod();
	a.somePrototypeMethod();
</script>

  • (this) and Looping

    You can also use this to fetch any value while iterating the loop.

    <script type="text/javascript">
	var person = [{
	  name: 'Sachin'
	},{
	  name: 'Ramesh'
	},{
	  name: 'Suresh'
	}];
	person.forEach(function (item) {
	  (function () {
		console.log(item); //Object {name: "Sachin"} 
		console.log(this.name); //Sachin
	  }).call(item);
	});
</script>
